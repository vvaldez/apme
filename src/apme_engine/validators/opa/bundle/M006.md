## Become timeout unreachable (M006)

In ansible-core 2.19+, a timeout waiting for privilege escalation (`become`) now raises an `UNREACHABLE` error instead of a task error. Tasks with `become: true` and `ignore_errors: true` will not catch the timeout unless `ignore_unreachable: true` is also set.

### Example: violation

```yaml
- name: Risky become task
  ansible.builtin.command: whoami
  become: true
  ignore_errors: true
```

### Example: pass

```yaml
- name: Safe become task
  ansible.builtin.command: whoami
  become: true
  ignore_errors: true
  ignore_unreachable: true
```
