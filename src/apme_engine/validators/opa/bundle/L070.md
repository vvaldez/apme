## Jinja position in task name (L070)

Dynamic task names should only have Jinja expressions at the end of the name string for readability.

### Example: violation

```yaml
- name: "Install {{ package }} on server"
  ansible.builtin.apt:
    name: "{{ package }}"
```

### Example: pass

```yaml
- name: "Install package {{ package }}"
  ansible.builtin.apt:
    name: "{{ package }}"
```
