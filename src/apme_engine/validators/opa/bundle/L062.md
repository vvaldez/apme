## YAML module args (L062)

Use YAML-style module arguments instead of `key=value` one-liners.

Detects `_raw_params` containing `=` in the OPA AST.

**Violation:** `ansible.builtin.apt: name=nginx state=present` — **Pass:** YAML mapping args
