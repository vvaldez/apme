## No Jinja in play names (L065)

Play names should be static strings, not Jinja templates.

### Example: violation

```yaml
- name: "Deploy {{ app_name }}"
  hosts: localhost
```

### Example: pass

```yaml
- name: Deploy application
  hosts: localhost
```
