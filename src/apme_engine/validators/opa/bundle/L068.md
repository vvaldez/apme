## Avoid lineinfile (L068)

Avoid `lineinfile` when alternatives exist. Prefer `template`, `ini_file`, `blockinfile`, or domain-specific modules.

### Example: violation

```yaml
- name: Set config value
  ansible.builtin.lineinfile:
    path: /etc/app.conf
    regexp: "^key="
    line: "key=value"
```

### Example: pass

```yaml
- name: Set config value
  ansible.builtin.template:
    src: app.conf.j2
    dest: /etc/app.conf
```
