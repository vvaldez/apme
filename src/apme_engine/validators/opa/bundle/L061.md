## Boolean format (L061)

Use `true`/`false` for booleans, not `yes`/`no`/`True`/`False`.

YAML parsers convert `yes`/`no` to boolean before the AST is built, so this rule operates on raw YAML input at the OPA level.

**Violation:** `enabled: yes` — **Pass:** `enabled: true`
