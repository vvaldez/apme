## Backup on copy/template (L072)

Consider setting `backup: true` on `template`/`copy` tasks so a backup of the original file is created before changes.

### Example: violation

```yaml
- name: Deploy config
  ansible.builtin.template:
    src: app.conf.j2
    dest: /etc/app.conf
```

### Example: pass

```yaml
- name: Deploy config
  ansible.builtin.template:
    src: app.conf.j2
    dest: /etc/app.conf
    backup: true
```
