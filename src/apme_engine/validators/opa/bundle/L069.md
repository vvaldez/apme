## Package loop (L069)

Package modules accept lists natively. Batch package names in the `name:` parameter instead of looping with `item`.

Detects package modules using `loop`/`with_items` with `item` in `name`.

**Violation:** `name: "{{ item }}"` with `loop:` — **Pass:** `name:` as a list
