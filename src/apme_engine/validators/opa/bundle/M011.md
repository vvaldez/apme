## Network collection compatibility (M011)

Network collections (`cisco.ios`, `cisco.nxos`, `cisco.iosxr`, `arista.eos`, `junipernetworks.junos`, `ansible.netcommon`) had compatibility issues with ansible-core 2.19 data tagging. Ensure these collections are updated to versions that support 2.19+.

### Example: violation

```yaml
- name: Get config
  cisco.ios.ios_command:
    commands: show running-config
```

### Example: pass

```yaml
- name: Get facts
  ansible.builtin.setup:
    gather_subset: all
```
