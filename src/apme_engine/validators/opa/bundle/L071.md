## Template over copy (L071)

When `copy` uses `content:` with Jinja expressions, prefer `template` module instead for clarity.

### Example: violation

```yaml
- name: Write config
  ansible.builtin.copy:
    content: "server={{ server_name }}"
    dest: /etc/app.conf
```

### Example: pass

```yaml
- name: Write config
  ansible.builtin.template:
    src: app.conf.j2
    dest: /etc/app.conf
```
