rule_id: L002
validator: opa
description: Use fully qualified collection name for modules.
---

## Use FQCN

Use fully qualified collection name for modules.

### Example: violation

```yaml
- name: Copy file
  my_custom_copy:
    src: foo
    dest: /tmp/foo
```

### Example: pass

```yaml
- name: Copy file
  ansible.builtin.copy:
    src: foo
    dest: /tmp/foo
```
