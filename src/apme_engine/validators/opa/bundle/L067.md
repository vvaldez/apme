## Debug verbosity (L067)

Set `verbosity` on `debug` tasks so messages are not always noisy in production.

### Example: violation

```yaml
- name: Show variable
  ansible.builtin.debug:
    var: my_var
```

### Example: pass

```yaml
- name: Show variable
  ansible.builtin.debug:
    var: my_var
    verbosity: 2
```
